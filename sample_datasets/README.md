# ProcessPilot AI Sample Datasets
This directory contains sample datasets to help you test and experience the capabilities of ProcessPilot AI.

You can upload these files in the **Documents** page or copy-paste transcripts into the **Meetings** page.

## Available Sample Files:

1. **`employee_roster.csv`** (CSV - HR/People Operations)
   - Contains an employee list with IDs, departments, roles, emails, and hire dates.
   - *Test query in Chat:* "List all software engineers in the engineering department" or "What is Sarah Jenkins' email?"

2. **`api_server_logs.txt`** (TXT - Engineering & DevOps Diagnostics)
   - Contains log trace entries showing API request responses, database connection pools, and error fallbacks.
   - *Test query in Chat:* "Did the server experience any rate limit errors on June 15?" or "Which LLM provider was used as a fallback when Groq failed?"

3. **`quarterly_sales_report.csv`** (CSV - Sales/Finance Data)
   - Contains product IDs, units sold in Q1/Q2, unit pricing, total revenues, and target sales regions.
   - *Test query in Chat:* "Which product generated the most revenue in Q1 and Q2?" or "How many units of training workshops were sold?"

4. **`meeting_transcript_marketing_sync.txt`** (TXT - Meeting Transcript & Alignment)
   - Contains a complete 15-minute transcript of a Q3 marketing strategy planning session.
   - *How to use:* Copy the text content and paste it on the **Meetings** page, or upload the file on the **Documents** page.
   - *Test query in Chat:* "Summarize the action items from the marketing sync meeting" or "What is Rohan's deadline for the landing page mockup?"
